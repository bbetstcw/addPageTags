### <span id="whathappened">我的项目发生了什么情况？</id>

##### 已添加引用

Azure 移动服务库已通过 **MobileServices.js** 文件形式添加到你的项目。
  
##### 移动服务的连接字符串值 

在 `services\mobileServices\settings` 文件夹中，已生成包含 **MobileServiceClient** 的新 JavaScript (.js) 文件，其中包含选定的移动服务的应用程序 URL 和应用程序密钥。
